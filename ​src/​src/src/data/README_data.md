# Data Availability Statement

Due to privacy and ethical considerations, the original hand injury surveillance data cannot be publicly shared. Researchers interested in accessing the data for verification or further study may contact the corresponding author (295256318@qq.com) with a formal request.

## Data Sources Description
- **Hand Injury Data**: Anonymized records from Hunan Provincial Institute
- **Meteorological Data**: Publicly available from China Meteorological Administration
- **Socioeconomic Data**: From Hunan Statistical Yearbook and China City Statistical Yearbook
- **Nighttime Light Data**: NASA's Earth Observing System
